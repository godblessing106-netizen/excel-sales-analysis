# 📊 Sales Data Analysis (Excel Project)

## 📌 Project Overview
This project analyzes sales performance across different products and regions to identify key business trends.

## 🎯 Objective
To understand:
- Which product generates the most revenue
- Which region performs best
- Overall sales distribution

## 🛠 Tools Used
- Microsoft Excel

## 📂 Dataset
The dataset contains:
- Date
- Product (Apple, Banana, Orange)
- Region (North, South, East, West)
- Sales values

## 📈 Key Insights
- Banana generated the highest revenue (41.1%)
- East region recorded the highest total sales (460)
- Orange contributed the lowest share of sales (25.8%)

## 📊 Analysis Performed
- Data aggregation using totals
- Percentage contribution analysis
- Regional performance comparison

## 📁 Files Included
- sales_data.xlsx

- How to summarize raw data into insights
- Basic business analysis thinking
- Presenting findings clearly
